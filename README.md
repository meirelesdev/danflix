# Danflix

> Clone do Netflix desenvolvido em React com React Router e Styled Components, consumindo a API do The Movie Database (TMDB).

## Stack

- React
- Styled Components

## Prerequisitos

- Node.js v14+

## Como executar

```bash
# Clone o repositorio
git clone https://github.com/meirelesdev/danflix.git
cd danflix

# Instale as dependencias
yarn install

# Inicie a aplicacao
npm start
```
